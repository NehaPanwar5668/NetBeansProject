**Project Description**
The Fees Management System is a Java-based desktop application developed using NetBeans IDE.
It is designed to manage student fee records efficiently, including adding student details, managing courses, recording payments, and tracking fee status.

**Features**
Add new student details
View and update student records
Add and manage courses
Record fee payments

**Technologies Used**
Java (Core Java)
Swing (for GUI)
MySQL (Database)
NetBeans IDE

**Learning Outcome**
Through this project, I improved my understanding of:
Java Swing GUI development
Database connectivity (JDBC)
CRUD operations
Error handling and debugging
